# Tesla Game Engine
Self learning project for understanding graphic API (OpenGL or Valkan) and the most important game engine!

### Implementation
* Render
  * VertexArray
  * Buffer
  * Renderer
  * Render command
  * Shader
  * FrameBuffer
  * Render pass
* Camera system
  * Move up/bottom/left/right
  * zoom in/out
* Event system
* Windows system
